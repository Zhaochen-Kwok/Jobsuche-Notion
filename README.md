# Jobsuche-Notion

This project is an automated job tracking tool built in Python to collect and organize relevant student job opportunities (e.g., internships and working student positions).

It retrieves job postings from a company’s web API, processes structured JSON data, and applies rule-based filtering using domain-specific keywords (e.g., AI, Data Analytics, Business). Relevant positions are then automatically stored in a Notion database via the Notion API, enabling structured tracking and deduplication.

The system allows continuous monitoring of new job postings and significantly improves the efficiency of job search and evaluation.
